Trabalho de Conclusão de Curso

O sistema foi desenvolvido utilizando React, Express e MySQL. Ele proporciona uma solução eficiente para o gerenciamento da biblioteca da escola, permitindo consultas de livros, reservas e acompanhamento de empréstimos. A interface React garante uma experiência de usuário fluida, enquanto a API com Express assegura a comunicação com o banco de dados MySQL, mantendo a integridade dos dados. Além disso, o sistema facilita o cadastro de novos livros, atualizações de informações e controle de devoluções e multas, otimizando a gestão da biblioteca.
